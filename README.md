# Implementação:

Implementação do algoritmo DSATUR para aplicação em grafos à partir de arquivos .dot

# Compilação:

Para compilar o programa, utilizar o seguinte comando:

g++ Dsatur.cpp -lboost_graph -lboost_regex -o dsatur && ./dsatur
